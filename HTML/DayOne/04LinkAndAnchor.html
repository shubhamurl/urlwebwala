1️⃣ What is the Tag? ✅ stands for anchor. ✅ It creates hyperlinks, which can: Navigate to another web page (internal or external) Download a file Open email clients Start phone calls Jump to sections on the same page (anchor links) 🟢 2️⃣ Basic Syntax Link Text href: The destination address. Link Text: The clickable text. 🟢 3️⃣ Types of Links Let’s see them one by one: ✅ A. Internal Links 👉 Link to another page within the same website. Example: About Us When you click this, it goes to: https://yourwebsite.com/about.html TIP: Use relative paths: about.html (same folder) folder/page.html (inside folder) ✅ B. External Links 👉 Link to another website. Example: Visit Example TIP: Always start with https:// or http:// for external URLs. ✅ C. Download Links 👉 Make a file download instead of opening. Example: Download Report ✅ download attribute tells the browser to download the file. Optionally, specify filename: Download Report When clicked, it saves as MyReport.pdf. ✅ D. Email Links (mailto:) 👉 Open the user’s default email app to send an email. Example: Email Us Add subject and body: Email with Subject ✅ Note: Spaces become %20 automatically. ✅ E. Phone Links (tel:) 👉 Make a click-to-call link (useful on mobile). Example: Call Us When clicked on a phone, it starts dialing the number. 🟢 4️⃣ Target Attribute The target attribute controls where the link opens: Value What it does _self Default – open in the same tab _blank Open in a new tab or window ✅ Example: Open in New Tab ✅ Tip: When using _blank, always add rel="noopener noreferrer" for security: Open Securely in New Tab 🟢 5️⃣ Anchor Links (Jump to a Section) 👉 These let you jump to another part of the same page or another page’s section. ✅ Step 1: Add id to the target element
